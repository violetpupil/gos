# [commands](https://kubernetes.io/docs/reference/generated/kubectl/kubectl-commands)

`get` 查看k8s资源

`run` 在pod中运行镜像

`delete` 删除k8s资源

## [通用选项]

`--namespace` 指定命名空间

`-A` 所有命名空间

## WORKING WITH APPS

`describe` 查看资源细节

## KUBECTL SETTINGS AND USAGE

`api-resources` 打印集群支持的api资源

`config` Modifies kubeconfig files.
