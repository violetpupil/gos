# id

## [shortid](https://pkg.go.dev/github.com/teris-io/shortid)

```golang
// id格式 H7yKLiMSR
shortid.Generate()
```

## [uuid](https://pkg.go.dev/github.com/google/uuid)

```golang
// Version 4 Random UUID 8b6b4b6a-7559-45a2-b0eb-80211671bf57
// 16 byte
uuid.NewRandom()
// 直接获取字符串
uuid.NewString()
```
